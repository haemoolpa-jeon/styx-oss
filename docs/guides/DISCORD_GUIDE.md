## 🚀 빠른 시작

### 1️⃣ 회원가입
1. 크롬 브라우저로 접속
2. "회원가입" 탭 선택
3. 이름(2~20자), 비밀번호(4자 이상) 입력
4. "가입 요청" 클릭
5. ⏳ 관리자 승인 대기

### 2️⃣ 로그인 후
1. 마이크/스피커 설정
2. "연결 테스트" 버튼으로 장비 확인
3. 방 만들기 또는 기존 방 입장

---

## 🎧 웹 버전 vs PC 앱

| | 웹 버전 | PC 앱 (Tauri) |
|---|---|---|
| **설치** | 필요 없음 | 설치 필요 |
| **지연** | 보통 (50~150ms) | **더 낮음** |
| **오디오** | 브라우저 기본 | **ASIO 지원** |
| **연결** | WebRTC | **UDP 직접 연결** |
| **안정성** | 좋음 | **더 좋음** |

### 💡 추천
- **일반 사용**: 웹 버전으로 충분
- **합주/녹음**: PC 앱 + 오디오 인터페이스 권장

---

## 💻 PC 앱 설치 (Windows)

### 다운로드
1. 설치 파일 다운로드 (NSIS 권장)
2. 실행하여 설치
3. 바탕화면 아이콘으로 실행

### UDP 모드 사용법
1. 대기실에서 "Custom UDP" 버튼 클릭
2. 방 입장 시 자동으로 UDP 연결
3. 상태바에서 연결 상태 확인

### ASIO 설정 (오디오 인터페이스 사용 시)
1. 대기실 → 오디오 호스트 → "ASIO" 선택
2. 입력/출력 장치 선택
3. 버퍼 크기는 낮을수록 저지연 (64~256 권장)

## 🎛️ 주요 기능

### 🎤 오디오
- **음성/음악 모드**: 용도에 맞게 선택
- **3밴드 EQ**: 저음/중음/고음 조절 (-12dB ~ +12dB)
- **입력 리미터**: 클리핑 자동 방지
- **개별 볼륨**: 참가자별 볼륨/팬/뮤트/솔로

### 🎬 화면 공유
- 🖥️ 버튼으로 악보/가사 공유
- 모든 참가자에게 실시간 표시

### 🎙️ 녹음
- **일반 녹음**: 전체 믹스를 하나의 파일로
- **멀티트랙**: 참가자별 개별 파일 (후반 작업용)

### 🎭 역할 시스템
| 역할 | 권한 |
|---|---|
| 👑 호스트 | 역할 변경, 방 관리 |
| 🎸 연주자 | 오디오 송수신 |
| 👂 청취자 | 듣기만 가능 |

### ⏱️ 메트로놈
- 30~300 BPM 설정
- 방 전체 동기화
- 4박 카운트인 지원

## 🔧 문제 해결

### ❌ 연결이 안 돼요
1. 페이지 새로고침 (F5)
2. 방 나갔다 다시 입장
3. 크롬 브라우저 확인
4. 방화벽에서 Styx 허용

### 🔇 소리가 안 들려요
1. 스피커/이어폰 연결 확인
2. 상대방 볼륨 슬라이더 확인
3. 브라우저 소리 권한 허용

### 🔊 소리가 끊겨요
1. **유선 연결 사용** (가장 중요!)
2. 버퍼 크기 올리기 (100~150ms)
3. "자동 지터 버퍼" 활성화
4. 다른 프로그램 종료

### 🐌 지연이 심해요
1. 버퍼 크기 내리기 (30~50ms)
2. PC 앱 + UDP 모드 사용
3. 유선 연결 확인

### 🔁 에코가 들려요
1. **이어폰/헤드폰 필수!**
2. "에코 제거" 설정 켜기
3. 마이크-스피커 거리 띄우기

## 🎯 합주 팁

### 지연 맞추기 (중요!)
1. "지연 맞추기" 옵션 활성화
2. 가장 느린 사람 기준으로 자동 동기화
3. 모두가 같은 타이밍에 듣게 됨

```
예시:
철수: 50ms → +100ms 보정 → 150ms
영희: 100ms → +50ms 보정 → 150ms
민수: 150ms → 그대로 → 150ms
→ 모두 150ms로 통일!
```

### 메트로놈 활용
1. 호스트가 BPM 설정
2. 카운트인 4박 후 시작
3. 서버 시간 동기화로 정확한 박자

### 녹음 팁
- 멀티트랙으로 녹음 → 나중에 DAW에서 믹싱
- 각자 로컬 녹음도 병행 권장
