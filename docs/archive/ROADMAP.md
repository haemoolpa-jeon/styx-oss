# Styx 개발 로드맵

## 현재 상태 (2025년 12월)

### ✅ 완료된 기능

**핵심 기능**
- 실시간 P2P 오디오 (WebRTC)
- 방 생성/입장 (비밀번호 지원)
- 채팅, 메트로놈 동기화
- 사용자 인증 및 관리자 승인

**오디오**
- Opus 코덱 최적화 (FEC, DTX)
- 음성/음악 모드
- 개인 믹서 (볼륨/팬/뮤트/솔로)
- PTT, VAD, 오디오 덕킹
- 세션 녹음

**연결 안정성**
- Coturn TURN 서버 (자체 호스팅)
- ICE candidate 큐잉
- 자동 재연결
- 서버 시간 동기화 (메트로놈)

**데스크톱 앱**
- Tauri + Rust 기반
- Custom UDP 모드 (저지연)
- ASIO 드라이버 지원
