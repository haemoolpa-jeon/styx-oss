## 서비스 소개

### 이런 분들을 위해 만들었어요
- 🎸 밴드 멤버들과 온라인으로 합주하고 싶은 분
- 🎤 실시간으로 음성 대화가 필요한 분
- 🎵 지연 없이 깔끔한 음질로 소통하고 싶은 분

### 어떻게 작동하나요?

```
┌─────────────────────────────────────────────────────────────────┐
│                         Styx 구조                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   ┌──────────┐         ┌──────────────┐         ┌──────────┐   │
│   │  내 PC   │◄───────►│   중앙 서버   │◄───────►│ 상대방 PC │   │
│   │(브라우저)│         │  (연결 중개)  │         │(브라우저)│   │
│   └────┬─────┘         └──────────────┘         └────┬─────┘   │
│        │                                              │         │
│        │         소리는 서버 안 거치고 직접 전달!        │         │
│        └──────────────────────────────────────────────┘         │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

**핵심 특징**
- 🚀 **빠른 연결**: 소리가 서버를 거치지 않고 바로 전달돼요
- 🎧 **좋은 음질**: 음악 모드에서 고음질 지원
- 🌐 **웹 브라우저로 바로 사용**: 설치 없이 크롬으로 접속
- ⏱️ **박자 맞추기**: 메트로놈과 지연 맞추기 기능 제공

## 처음 시작하기

### 접속하기
**크롬 브라우저**로 아래 주소에 접속하세요:

👉 **https://3-39-223-2.nip.io**

> 💡 **크롬 브라우저를 권장합니다.** 다른 브라우저에서는 일부 기능이 제한될 수 있어요.

### 계정 만들기
1. 화면 아래 "회원가입" 탭 선택
2. 사용할 이름 입력 (2~20자, 한글/영문/숫자 가능)
3. 비밀번호 입력 (4자 이상)
4. "가입 요청" 버튼 클릭
5. ⏳ **관리자가 승인하면 로그인할 수 있어요**

### 로그인하기
1. 이름과 비밀번호 입력
2. "로그인" 버튼 클릭
3. 대기실로 이동!

## 대기실 사용법

### 내 프로필 꾸미기
- **사진 바꾸기**: 프로필 사진 클릭 → 원하는 이미지 선택 (2MB 이하)
- **나가기**: 오른쪽 위 로그아웃 버튼

### 마이크/스피커 설정
| 항목 | 설명 |
|------|------|
| 입력 장치 | 사용할 마이크 선택 |
| 출력 장치 | 소리 들을 스피커/이어폰 선택 |
| 테스트 | 마이크가 잘 되는지 확인 |

### 방 목록 보기
- 현재 열려있는 방들이 보여요
- 🔒 표시: 비밀번호가 있는 방
- 👤 숫자: 지금 몇 명 / 최대 8명
- ✕ 버튼: 방 삭제 (방 만든 사람이나 관리자만)

### 새 방 만들기
1. "방 만들기" 버튼 클릭
2. 방 이름 입력
3. 비밀번호 설정 (안 해도 됨)
4. "만들기" 클릭

## 방 안에서

### 기본 버튼들
| 버튼 | 하는 일 |
|------|--------|
| 🎤 / 🔇 | 마이크 켜기/끄기 |
| 📋 | 초대 링크 복사 (친구에게 보내세요) |
| ⏺️ | 녹음 시작/멈추기 |
| 🚪 | 방 나가기 |
| ❌ | 방 닫기 (방장/관리자용) |

### 채팅하기
- 오른쪽 채팅창에서 글로 대화할 수 있어요
- Enter 키로 보내기

### 메트로놈 (박자기)
1. 원하는 BPM 입력 (30~300)
2. ▶️ 버튼으로 시작
3. **방에 있는 모든 사람에게 같은 박자가 들려요**

### 참가자 정보 보기
각 사람 카드에서 확인할 수 있는 것들:
- 프로필 사진과 이름
- 연결 상태 (연결 중.../연결됨)
- 연결 품질 (좋음 🟢 / 보통 🟡 / 불안정 🔴)
- 지연 시간 (숫자가 작을수록 좋아요)
- 소리 크기 조절 슬라이더

## 소리 설정

### 방 안에서 바꿀 수 있는 설정

| 설정 | 무슨 기능? | 언제 쓰면 좋아요? |
|------|-----------|-----------------|
| **눌러서 말하기** | Space 키 누를 때만 마이크 켜짐 | 주변이 시끄러울 때 |
| **말할 때 표시** | 말하는 사람 표시해줌 | 항상 켜두세요 |
| **자동 품질 조절** | 인터넷 상태 따라 음질 자동 조절 | 항상 켜두세요 |
| **자동 볼륨 낮춤** | 다른 사람 말할 때 내 소리 줄여줌 | 여러 명일 때 |
| **지연 맞추기** | 모든 사람 타이밍 맞춰줌 | 합주할 때 필수! |

### 버퍼 크기 조절
소리가 끊기면 올리고, 지연이 심하면 내리세요.
- **50ms**: 가장 빠름, 끊길 수 있음
- **100ms**: 기본값, 대부분 OK
- **200ms**: 가장 안정적, 약간 느림

### 소리 모드 (대기실에서 설정)
| 모드 | 음질 | 용도 |
|------|------|------|
| 🎤 대화 모드 | 보통 | 일반 대화 |
| 🎵 음악 모드 | 고음질 | 악기 연주, 노래 |

## 유용한 기능

### 🎯 지연 맞추기
합주할 때 가장 중요한 기능이에요!

**왜 필요해요?**
- 인터넷 속도가 사람마다 달라요
- A는 0.05초, B는 0.15초 늦게 들릴 수 있어요
- 이러면 박자가 안 맞아요!

**어떻게 해결해요?**
- 가장 느린 사람 기준으로 맞춰요
- 빠른 사람들은 일부러 조금 늦춰요
- 결과: 모두가 같은 타이밍에 들어요!

**예시:**
```
철수: 50ms 지연  → 100ms 더 늦춤 → 150ms
영희: 100ms 지연 → 50ms 더 늦춤  → 150ms  
민수: 150ms 지연 → 그대로        → 150ms

→ 세 사람 모두 150ms로 통일!
```

### 👑 관리자 기능
- 새 회원 가입 승인/거절
- 회원 삭제
- 모든 방 관리

## 자주 묻는 질문

### 연결이 안 돼요
1. 페이지 새로고침 (F5)
2. 방에서 나갔다 다시 들어가기
3. 크롬 브라우저인지 확인

### 상대방 소리가 안 들려요
1. 내 스피커/이어폰 확인
2. 상대방 카드의 볼륨 슬라이더 확인
3. 브라우저에서 소리 권한 허용했는지 확인

### 소리가 뚝뚝 끊겨요
1. 버퍼 크기 올리기 (100~150ms) 또는 "자동" 체크
2. 자동 품질 조절 켜기
3. **Wi-Fi 대신 유선 인터넷 사용 (매우 중요!)**

### 지연이 너무 심해요
1. 버퍼 크기 내리기 (30~50ms)
2. 유선 인터넷 사용

### 내 소리가 울려요 (에코)
1. 이어폰/헤드폰 사용 (스피커 X)
2. 대기실 설정에서 "에코 제거" 켜기
3. 마이크와 스피커 거리 띄우기

## 새로운 기능 (2025.12)

### 🎬 화면 공유
- 🖥️ 버튼으로 화면 공유 시작
- 악보, 가사, 코드 등을 모든 참가자에게 표시
- 공유 중지하려면 다시 클릭

### 🎭 역할 시스템
- **호스트**: 방 생성자, 다른 사람 역할 변경 가능
- **연주자**: 오디오 송수신 가능 (기본값)
- **청취자**: 듣기만 가능, 마이크 비활성화

### 🎛️ 3밴드 EQ
- 🎛️ 버튼으로 EQ 패널 열기
- 저음/중음/고음 각각 -12dB ~ +12dB 조절
- 설정은 자동 저장됨

### 🎙️ 멀티트랙 녹음
- "멀티트랙" 체크박스 활성화
- 녹음 시 각 참가자별로 개별 파일 저장
- 후반 작업(믹싱)에 유용

### 📡 네트워크 품질 측정
- 연결 테스트 시 지연/지터 측정
- Wi-Fi 사용 시 경고 메시지 표시
- 권장: **유선 연결 사용**

### 🔄 자동 지터 버퍼
- "자동" 체크박스 활성화
- 네트워크 상태에 따라 30-150ms 자동 조절
- 끊김과 지연 사이 최적 균형 유지

## PC 앱 출시 예정

### 🖥️ 전용 앱이 곧 나와요!

현재는 웹 버전만 사용 가능하지만, **Windows/Mac용 전용 앱**을 준비 중이에요.

### 웹 버전 vs PC 앱 비교

| 기능 | 웹 버전 | PC 앱 (예정) |
|------|--------|-------------|
| 설치 | 필요 없음 | 설치 필요 |
| 접속 | 브라우저로 바로 | 앱 실행 |
| 지연 시간 | 보통 | **더 빠름** |
| 오디오 장치 | 기본 지원 | **ASIO 등 전문 장비 지원** |
| 연결 방식 | WebRTC | **UDP 직접 연결 (더 빠름)** |

### PC 앱에서 추가될 기능
- ⚡ **초저지연 UDP 모드**: 웹보다 더 빠른 연결
- 🎛️ **전문 오디오 장비 지원**: ASIO 드라이버, 오디오 인터페이스
- 🔧 **세밀한 설정**: 오디오 버퍼, 샘플레이트 조절
- 📊 **연결 테스트**: 내 네트워크 상태 상세 확인

### 출시 일정
- 현재: 웹 버전 운영 중
- 추후: Windows/Mac 앱 출시 예정

> 💡 PC 앱이 출시되면 공지해 드릴게요. 그때까지 웹 버전을 이용해 주세요!
