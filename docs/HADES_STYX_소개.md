# Styx 소개 (HADES 멤버용)

## 왜 직접 만들었나?

### SyncRoom (야마하) 문제점
- 일본 서버 위주라 한국에서 지연이 큼
- 무료 버전은 기능 제한
- 불특정 다수와 트래픽 경쟁

### SonoBus 문제점
- 설정이 복잡함
- UI가 직관적이지 않음

## Styx의 장점

### 1. 소규모 전용 = 쾌적한 환경
- 최대 8명 (HADES 전용)
- 서버 부하 거의 없음
- 트래픽 경쟁 없음

### 2. 지연 시간
- 목표: 30~50ms (네트워크 상태에 따라 다름)
- TURN 서버로 NAT/방화벽 뒤에서도 연결 가능

> 💡 사람이 "동시에 연주한다"고 느끼려면 30ms 이하가 이상적
> 50ms까지는 적응 가능, 100ms 이상은 어려움

### 3. 개인별 설정
- 에코/노이즈 제거 ON/OFF
- PTT (Push-to-Talk) 모드
- 개인별 볼륨/팬 조절
- 음성/음악 모드 선택

## 자주 묻는 질문

**Q: 인터넷 속도가 느려도 되나요?**
> 10Mbps 이상이면 충분합니다.

**Q: 오디오 인터페이스가 있으면 더 좋나요?**
> 데스크톱 앱에서 ASIO 드라이버를 사용하면 지연이 더 줄어듭니다.
> 없어도 사용 가능합니다.

**Q: 연결이 안 돼요**
> 1. 브라우저 새로고침
> 2. 마이크 권한 확인
> 3. 다른 브라우저(크롬 권장) 시도
