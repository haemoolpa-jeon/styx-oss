[ 서비스 소개 ]

Styx는 밴드 멤버들과 온라인으로 합주하거나, 실시간 음성 대화가 필요한 분들을 위해 만들었어요. 지연 없이 깔끔한 음질로 소통할 수 있습니다.

소리가 서버를 거치지 않고 사용자끼리 직접 전달되기 때문에 빠르고, 음악 모드에서는 고음질도 지원해요. 설치 없이 크롬 브라우저로 바로 사용할 수 있고, 메트로놈과 지연 맞추기 기능도 있어요.


───────────────────────────────────────────────────────────────
[ 처음 시작하기 ]
───────────────────────────────────────────────────────────────

★ 접속하기
크롬 브라우저로 아래 주소에 접속하세요.
→ https://3-39-223-2.nip.io

(크롬 브라우저를 권장합니다. 다른 브라우저에서는 일부 기능이 제한될 수 있어요.)


★ 계정 만들기
1. 화면 아래 "회원가입" 탭 선택
2. 사용할 이름 입력 (2~20자, 한글/영문/숫자 가능)
3. 비밀번호 입력 (4자 이상)
4. "가입 요청" 버튼 클릭
5. 관리자가 승인하면 로그인할 수 있어요


★ 로그인하기
이름과 비밀번호 입력 후 "로그인" 버튼을 클릭하면 대기실로 이동합니다.


───────────────────────────────────────────────────────────────
[ 대기실 사용법 ]
───────────────────────────────────────────────────────────────

★ 내 프로필 꾸미기
프로필 사진을 클릭하면 원하는 이미지로 바꿀 수 있어요. (2MB 이하)
오른쪽 위 버튼으로 로그아웃할 수 있습니다.


★ 마이크/스피커 설정
- 입력 장치: 사용할 마이크 선택
- 출력 장치: 소리 들을 스피커/이어폰 선택
- 테스트 버튼으로 마이크가 잘 되는지 확인할 수 있어요


★ 방 목록 보기
현재 열려있는 방들이 보여요.
- 자물쇠 표시가 있으면 비밀번호가 있는 방이에요
- 숫자는 현재 인원 / 최대 8명을 뜻해요
- X 버튼은 방 만든 사람이나 관리자만 누를 수 있어요


★ 새 방 만들기
"방 만들기" 버튼 클릭 → 방 이름 입력 → 비밀번호 설정(선택) → "만들기" 클릭


───────────────────────────────────────────────────────────────
[ 방 안에서 ]
───────────────────────────────────────────────────────────────

★ 기본 버튼들
- 마이크 버튼: 마이크 켜기/끄기
- 클립보드 버튼: 초대 링크 복사 (친구에게 보내세요)
- 녹음 버튼: 녹음 시작/멈추기
- 문 버튼: 방 나가기
- X 버튼: 방 닫기 (방장/관리자용)


★ 채팅하기
오른쪽 채팅창에서 글로 대화할 수 있어요. Enter 키로 보내기.


★ 메트로놈 (박자기)
원하는 BPM을 입력하고 재생 버튼을 누르면 방에 있는 모든 사람에게 같은 박자가 들려요.


★ 참가자 정보 보기
각 사람 카드에서 프로필 사진, 이름, 연결 상태, 연결 품질, 지연 시간을 확인할 수 있어요. 슬라이더로 상대방 소리 크기도 조절할 수 있습니다.


───────────────────────────────────────────────────────────────
[ 소리 설정 ]
───────────────────────────────────────────────────────────────

★ 방 안에서 바꿀 수 있는 설정

- 눌러서 말하기: Space 키 누를 때만 마이크가 켜져요. 주변이 시끄러울 때 유용해요.

- 말할 때 표시: 말하는 사람을 표시해줘요. 항상 켜두세요.

- 자동 품질 조절: 인터넷 상태에 따라 음질을 자동으로 조절해요. 항상 켜두세요.

- 자동 볼륨 낮춤: 다른 사람이 말할 때 내 소리를 줄여줘요. 여러 명일 때 유용해요.

- 지연 맞추기: 모든 사람의 타이밍을 맞춰줘요. 합주할 때 필수!


★ 버퍼 크기 조절
소리가 끊기면 올리고, 지연이 심하면 내리세요.
- 50ms: 가장 빠름, 끊길 수 있음
- 100ms: 기본값, 대부분 OK
- 200ms: 가장 안정적, 약간 느림


★ 소리 모드 (대기실에서 설정)
- 대화 모드: 일반 대화용, 보통 음질
- 음악 모드: 악기 연주나 노래할 때, 고음질


───────────────────────────────────────────────────────────────
[ 지연 맞추기 기능 ]
───────────────────────────────────────────────────────────────

합주할 때 가장 중요한 기능이에요!

인터넷 속도가 사람마다 달라서, 어떤 사람은 0.05초, 어떤 사람은 0.15초 늦게 들릴 수 있어요. 이러면 박자가 안 맞죠.

이 기능을 켜면 가장 느린 사람 기준으로 맞춰요. 빠른 사람들은 일부러 조금 늦춰서 모두가 같은 타이밍에 듣게 됩니다.

예를 들어 철수가 50ms, 영희가 100ms, 민수가 150ms 지연이라면, 철수에게 100ms, 영희에게 50ms를 더해서 세 사람 모두 150ms로 맞춰요.


───────────────────────────────────────────────────────────────
[ 자주 묻는 질문 ]
───────────────────────────────────────────────────────────────

Q. 연결이 안 돼요
→ 페이지 새로고침(F5) 해보세요. 안 되면 방에서 나갔다 다시 들어가보세요. 크롬 브라우저인지도 확인해주세요.

Q. 상대방 소리가 안 들려요
→ 내 스피커/이어폰 확인, 상대방 카드의 볼륨 슬라이더 확인, 브라우저에서 소리 권한 허용했는지 확인해보세요.

Q. 소리가 뚝뚝 끊겨요
→ 버퍼 크기를 150~200ms로 올려보세요. 자동 품질 조절도 켜두시고, 가능하면 와이파이 대신 유선 인터넷을 사용해보세요.

Q. 지연이 너무 심해요
→ 버퍼 크기를 50~100ms로 내려보세요. 유선 인터넷 사용을 권장합니다.

Q. 내 소리가 울려요 (에코)
→ 이어폰이나 헤드폰을 사용하세요. 스피커로 들으면 에코가 생길 수 있어요. 대기실 설정에서 "에코 제거"도 켜보세요.

[ PC 앱 출시 예정 ]
───────────────────────────────────────────────────────────────

현재는 웹 버전만 사용 가능하지만, Windows/Mac용 전용 앱을 준비 중이에요.

PC 앱이 나오면 이런 점이 좋아져요:
- 웹보다 더 빠른 연결 (초저지연 UDP 모드)
- ASIO 드라이버 등 전문 오디오 장비 지원
- 오디오 버퍼, 샘플레이트 등 세밀한 설정
- 내 네트워크 상태 상세 확인

PC 앱이 출시되면 공지해 드릴게요. 그때까지 웹 버전을 이용해 주세요!

[ 기술 정보 ]
───────────────────────────────────────────────────────────────

- 소리 형식: Opus 코덱
- 음질: 48kHz, 32~128kbps
- 최대 인원: 방당 8명
- 연결 방식: P2P (직접 연결)
- 권장 브라우저: Chrome (크롬)
