# 🗺️ Styx 개발 로드맵

## 현재 상태 (2024년 12월)

### ✅ 완료된 기능

**웹 버전 (v2.3)**
- 실시간 음성 통화 (WebRTC)
- 방 생성/입장 (비밀번호 지원)
- 채팅, 메트로놈 동기화
- 오디오 설정 (에코/노이즈 제거, 음성/음악 모드)
- PTT, VAD, 오디오 덕킹
- 연결 품질 대시보드, 핑 그래프
- 세션 녹음, 초대 링크
- 다크/라이트 테마

**데스크톱 앱 (Phase 1-11)**
- Tauri + Rust 기반
- Custom UDP 모드 (저지연)
- Opus 코덱 (48kHz mono)
- STUN/TURN 지원
- 지터 버퍼 (50ms)
- ASIO 드라이버 감지
- 연결 품질 모니터링
