# 🧪 Styx 테스트 가이드

테스트에 참여해 주셔서 감사합니다! 이 문서는 테스트 방법과 피드백 제출 방법을 안내합니다.

---

## 📋 테스트 전 준비

### 필수 장비
- ✅ 헤드폰 또는 이어폰 (스피커 사용 시 에코 발생)
- ✅ 마이크 (내장 마이크 가능)
- ✅ 인터넷 연결 (유선 권장, 10Mbps 이상)

### 권장 장비
- 🎧 오디오 인터페이스 + ASIO 드라이버 (저지연)
- 🔌 유선 인터넷 (와이파이보다 안정적)

---

## 🌐 웹 버전 테스트

### 접속 방법
1. 브라우저에서 서버 주소 접속
2. 회원가입 (관리자 승인 필요)
3. 로그인 후 방 생성 또는 입장

### 테스트 항목

#### 1. 기본 기능
- [ ] 로그인/로그아웃
- [ ] 방 생성 (비밀번호 있음/없음)
- [ ] 방 입장/퇴장
- [ ] 채팅 전송/수신

#### 2. 오디오
- [ ] 마이크 입력 확인 (레벨 미터)
- [ ] 상대방 소리 들림
- [ ] 음소거 버튼 작동
- [ ] 볼륨 조절 작동
- [ ] PTT 모드 작동 (Space 키)

#### 3. 설정
- [ ] 마이크 장치 변경
- [ ] 스피커 장치 변경
- [ ] 음성/음악 모드 전환
- [ ] 에코 제거 ON/OFF
- [ ] 노이즈 제거 ON/OFF

#### 4. 안정성
- [ ] 10분 이상 연결 유지
- [ ] 네트워크 끊김 후 재연결
- [ ] 여러 명 동시 접속 (2-4명)

## 🖥️ 데스크톱 앱 테스트

### 설치 방법
1. [릴리즈 페이지](링크)에서 설치 파일 다운로드
2. `styx-desktop_x.x.x_x64-setup.exe` 실행
3. 설치 완료 후 실행

### 테스트 항목

#### 1. 설치/실행
- [ ] 설치 정상 완료
- [ ] 앱 실행됨
- [ ] 시스템 트레이 아이콘 표시
- [ ] 창 닫기 시 트레이로 최소화

#### 2. WebRTC 모드
- [ ] 웹 버전과 동일하게 작동
- [ ] 연결 품질 표시

#### 3. Custom UDP 모드 (저지연)
- [ ] UDP 모드 선택 가능
- [ ] STUN으로 공인 IP 획득
- [ ] 피어 연결 성공
- [ ] 오디오 송수신

#### 4. ASIO (오디오 인터페이스 사용자)
- [ ] ASIO 드라이버 감지됨
- [ ] ASIO 선택 가능

## 📊 성능 테스트

### 지연 시간 측정
1. 설정 → 연결 테스트 실행
2. 핑 그래프 확인
3. 결과 스크린샷 공유

### 측정 항목
| 항목 | 좋음 | 보통 | 나쁨 |
|------|------|------|------|
| 핑 | <50ms | 50-100ms | >100ms |
| 패킷 손실 | <1% | 1-5% | >5% |
