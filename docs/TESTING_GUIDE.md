# 테스트 가이드

## 테스트 전 준비

### 필수 장비
- ✅ 헤드폰 또는 이어폰 (스피커 사용 시 에코 발생)
- ✅ 마이크 (내장 마이크 가능)
- ✅ 인터넷 연결 (유선 권장)

### 권장 장비
- 🎧 오디오 인터페이스 + ASIO 드라이버 (저지연)
- 🔌 유선 인터넷

## 웹 버전 테스트

### 접속
1. 크롬 브라우저에서 서버 주소 접속 (예: `http://localhost:3000`)
2. 회원가입 (관리자 승인 필요)
3. 로그인 후 방 생성 또는 입장

### 테스트 항목

#### 기본 기능
- [ ] 로그인/로그아웃
- [ ] 방 생성 (비밀번호 있음/없음)
- [ ] 방 입장/퇴장
- [ ] 채팅 전송/수신

#### 오디오
- [ ] 마이크 입력 확인 (레벨 미터)
- [ ] 상대방 소리 들림
- [ ] 음소거 버튼 작동
- [ ] 볼륨 조절 작동

#### 안정성
- [ ] 10분 이상 연결 유지
- [ ] 네트워크 끊김 후 재연결
- [ ] 여러 명 동시 접속 (2-4명)

## 성능 측정

| 항목 | 좋음 | 보통 | 나쁨 |
|------|------|------|------|
| 핑 | <50ms | 50-100ms | >100ms |
| 패킷 손실 | <1% | 1-5% | >5% |
